## Run Locally 🚀

- git clone : git@github.com:AbdulWahab0/Wunderflats-Automation-Task.git
- cd Wunderflats-Automation-Task
- npm install
- npm start
- Click on feature file  (ProfileTab.feature) to run the Automated test
